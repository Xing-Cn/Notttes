# IDA
*快捷键*

### 通用导航与视图

| 快捷键     | 功能说明                               |
|-----------|----------------------------------------|
| `Space`   | 反汇编视图 <-> 图结构视图 切换           |
| `g`       | 跳转到任意地址 / 符号                  |
| `Esc`     | 返回到上一次跳转前的位置                |
| `Ctrl+F`  | 在当前视图中搜索 文本/指令              |
| `Shift+F12` | 打开字符串窗口(Strings), 搜索字符串   |
| `Alt+T`   | 搜索程序中的 指令/文本                  |

---

### 反汇编与伪代码

| 快捷键     | 功能说明                               |
|-----------|----------------------------------------|
| `F5`      | 生成/切换 到 C 伪代码视图               |
| `;`       | 在反汇编视图中为当前指令添加注释         |
| `/`       | 在伪代码视图中添加注释                  |
| `n`       | 定义或修改名称(函数, 变量, 标签等)       |
| `Y`       | 修改 变量/函数 类型(函数 原型/结构体 等) |
| `Ctrl+X`  | 查看当前符号(变量/函数)的引用(xrefs)     |

---

### 数据视图与显示格式

| 快捷键     | 功能说明                               |
|-----------|----------------------------------------|
| `A`       | 按 ASCII 字符串显示当前数据              |
| `D`       | 定义为数据(按 字节/字/双字 等循环切换)    |
| `R`       | 将当前数据解释为 数/数值 类型            |
| `*`       | Convert to array, 将当前数据转换为数组   |

---

### 调试相关

| 快捷键     | 功能说明                               |
|-----------|----------------------------------------|
| `F2`      | 在当前行快速 设置/取消 断点              |
