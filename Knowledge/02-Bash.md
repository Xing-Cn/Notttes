# Bash
*基础操作*

> Bash(Bourne-Again SHell) 是 `Linux/Unix` 系统中最常见的命令行解释器  
> 可以用来执行命令, 编写脚本, 进行自动化操作

---

### 查看文件内容

| 命令                     | 功能说明                    |
|--------------------------|-----------------------------|
| `cat file.txt`           | 显示整个文件                |
| `head -n 10 file.txt`    | 显示前 10 行                |
| `tail -n 10 file.txt`    | 显示后 10 行                |
| `less file.txt`          | 分页查看(按 `q` 退出)        |

---

### 文件信息查看

| 命令                     | 功能说明                          |
|--------------------------|-----------------------------------|
| `file flag.txt`          | 查看文件类型                      |
| `ls -la`                 | 显示详细文件列表(包含隐藏文件)      |
| `stat flag.txt`          | 查看文件详细信息                  |
| `wc -l file.txt`         | 统计行数                          |

---

### 搜索文件内容

| 命令                         | 功能说明                            |
|------------------------------|-------------------------------------|
| `grep "flag{" file.txt`      | 搜索包含 `"flag{"` 的行            |
| `grep -r "password" .`       | 递归搜索当前目录中的 `"password"`  |
| `strings binary_file`        | 提取二进制文件中的可打印字符串      |
| `hexdump -C file.bin`        | 十六进制查看(带 ASCII 显示)        |
| `xxd file.bin`               | 另一种常用的十六进制查看工具        |
